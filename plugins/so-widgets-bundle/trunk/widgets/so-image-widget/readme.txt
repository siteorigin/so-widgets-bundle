A simple widget for adding images from your media library to your sidebars, widgetized areas or Page Builder pages.

== Description ==

This plugin gives you a simple image widget that lets you add images from your media library. You can choose image size, destination URL as well as some other important SEO attributes like alt and title tags.

[Image Widget documentation](http://siteorigin.com/image-widget-documentation/) is available on SiteOrigin
