== Description ==

One very popular trend in websites over the last few years is to have a list of product or service features displayed as a series of icons. This widget makes those easy to build with drag and drop widget interface.

You can add these to your Page Builder pages or any of your theme's widgetized areas. There are ample icon, shapes and color settings to help you customize this widget to suit your site's design.

Documentation is available on [SiteOrigin](http://siteorigin.com/features-widget-documentation/).
