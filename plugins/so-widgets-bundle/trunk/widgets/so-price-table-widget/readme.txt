Provides a price table widget to display pricing options for your product or service.

== Description ==

The price table widget gives you an easy drag and drop interface to create price tables. You can add these to any widgetized area or [Page Builder](http://wordpress.org/plugins/siteorigin-panels/) page.

For help setting up your price table, read the [documentation](http://siteorigin.com/price-table-widget/) on SiteOrigin.

The price table widget has 4 primary colour customization options so you can make it uniquely your own. Over time we'll be adding more themes, so you can choose one that suits your personal preference.

= Icons =

The price table gives you access to all the current Font Awesome icons. Font Awesome is licensed under [SIL OFL 1.1](http://fontawesome.io/license/).
