Gives you a widget to display your posts as a carousel.

== Description ==

The price table widget gives you an easy interface create post carousel. You can add these to any widgetized area or [Page Builder](http://wordpress.org/plugins/siteorigin-panels/) page.

The widget has a post query builder that lets you select posts by type and taxonomy, or you can select specific posts to display.
