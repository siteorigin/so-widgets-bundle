A simple widget for adding buttons to your sidebars, widgetized areas or Page Builder pages.

== Description ==

[youtube http://www.youtube.com/watch?v=mPAgVUIFCSs]

[Full documentation](http://siteorigin.com/button-widget-documentation/) is available on SiteOrigin.

This plugin gives you a simple button widget that you can add to your widgetized area or [Page Builder](http://wordpress.org/plugins/siteorigin-panels/) pages.

Here are some of the features that make this button widget, particularly useful.

* Unlimited color customizations.
* 1500+ bundled button icons from packs like Font Awesome, Genericons and others.
* 3 unique button styles
* Change padding, font size and button rounding for some really unique looking buttons
